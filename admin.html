🚀 Publish Data to Website

Your data (results, teachers, activities) is stored locally in your browser. To make it visible to everyone on the internet, you must publish it by following these steps:

📋 How to Publish (do this every time you update data)

  1. Click the "📥 Download data.json" button below — it downloads a file called data.json
  2. Go to your GitHub repository (where the website files are hosted)
  3. Find the existing data.json file in the repo root and click it
  4. Click the pencil (edit) icon or use "Upload file" to replace it with the new one
  5. Click "Commit changes" — GitHub Pages will update in 1–2 minutes
  6. Refresh the website on any device — everyone will now see the updated data ✅
💡 GitHub Upload Tip: In your GitHub repo → click data.json → click ✏️ Edit → paste the JSON content → Commit. Or: drag the downloaded file onto the repo page and commit.
